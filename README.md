# Benji's Web Solutions

A modern, single-page marketing site built with [Vite](https://vitejs.dev/).
Fast, responsive, and ready to deploy on [Vercel](https://vercel.com/).

## Development

```bash
npm install      # install dependencies
npm run dev      # start the dev server (http://localhost:5173)
npm run build    # build for production -> dist/
npm run preview  # preview the production build locally
```

## Deploying to Vercel

This is a standard Vite project, so Vercel auto-detects it:

- **Framework preset:** Vite
- **Build command:** `npm run build`
- **Output directory:** `dist`

Just import the GitHub repository into Vercel and deploy — no extra
configuration required.

## Tech

- Vite (vanilla JS, zero runtime framework)
- Modern CSS (custom properties, gradients, glassmorphism, scroll reveals)
- No build-time dependencies beyond Vite
